===========================
###########功能描述
提供 cocos 下载资源功能封装
与原生封装差异性

1.支持数组下载

2.支持直接下载龙骨动画

3.支持按照分包卸载资源

4.增加加载过程事件抛发机制

5.加载过程中无需判断是否已经完成 bundle 加载

###########环境依赖
cocos2.4 环境

###########发布步骤
1.npm run build:all

2.修改 package.json 版本号

3.npm publish

###########运行部署步骤

1. npm install zmg_res_mgr
